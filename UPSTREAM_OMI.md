# Upstream Omi Relationship

Omi Ambient Companion began as an experimental companion-app branch inside a fork of `BasedHardware/omi`.

This repository intentionally contains only the standalone Android companion app, the optional Ambient Second Brain Controller plugin backend, and the minimal CI/docs needed to build and test those pieces.

The companion integrates with the official Omi service through user-authenticated Omi API calls. It does not modify or replace the official Omi app.
